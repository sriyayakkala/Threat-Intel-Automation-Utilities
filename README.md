 Threat Intelligence & OSINT Automation Utilities

A collection of lightweight, production-ready Python utilities designed to streamline Cyber Threat Intelligence (CTI) workflows, automate Indicator of Compromise (IoC) extraction, and accelerate open-source intelligence gathering.

Tools Included

 1. Breach Data & Log Parser (`log_parser.py`)
This utility automates the ingestion and parsing of unstructured text data (such as raw dark web credential leaks or massive network server logs). Utilizing high-performance Regular Expressions (Regex), it instantly strips out and structures key operational data points.
*   **Target Indicators:** Extracts validated Email Addresses and IPv4 Addresses.
*   **Output:** Generates a structured `.csv` matrix mapping the extracted threat indicators.

### 2. Automated OSINT Threat Scraper (`osint_scraper.py`)
A command-line tool built to eliminate manual pivoting during asset triaging. It programmatically interfaces with public threat databases to evaluate the risk score and geographical footprint of a target infrastructure asset.
*   **Core Logic:** Queries threat API nodes for real-time validation.
*   **Output:** Compiles raw technical response payloads into a clean, executive-level Terminal Threat Report.
